# CarneShop 🦔

> Sistema de Delivery de Carnes SaaS - Plataforma multi-tenant para açougues e churrascarias

## 📋 Documentação

Consulte o arquivo [SPEC.md](./SPEC.md) para todas as especificações do projeto.

## 🚀 Stack Tecnológica

| Camada | Tecnologia |
|--------|------------|
| Backend | Laravel 11 (PHP 8.3+) |
| Frontend | React Native (Expo) |
| Banco de Dados | PostgreSQL 15+ |
| Cache/Queue | Redis |

## 📦 Módulos

- **SaaS Core** - Gestão de tenants, planos, assinaturas
- **Loja** - Gestão de produtos, categorias, pedidos
- **Cliente** - Catálogo, carrinho, checkout via WhatsApp
